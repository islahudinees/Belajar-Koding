Inline
- Elemen HTML yang secara default tidak menambkan baris baru ketika dibuat
- Lebar dan tinggi elementnya sebesar yang ada didalamnya
- Kita tidak dapat mengatur tinggi dan lebar dari elemen inline
- Margin dan padding hanya mempengaruhi element secara herizontal, tidak vertikal
